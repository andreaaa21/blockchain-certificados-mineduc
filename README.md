# Sistema de Certificacion de Ingles con Blockchain
### Ministerio de Educacion de Guatemala - MINEDUC

Prototipo para emitir y verificar certificados de nivel de ingles
al graduarse del diversificado.

## Tecnologias utilizadas
- Python 3 (Google Colab)
- hashlib — hashes SHA-256
- fpdf2 — generacion de PDFs
- qrcode — codigos QR

## Como ejecutar
1. Abrir el archivo .ipynb en Google Colab
2. Ejecutar las celdas en orden
3. Los PDFs se generan en /content/ de Colab
